# Criador-de-coelhos
Um sistema em Java feito para catalogar coelhos, suas crias, compradores e vendas.

IDE: Apache NetBeans versão 22

Recomendações: 

- Ter conexão com servidores Apache e MySQL. De outra maneira, o programa é virtualmente inútil.
- Seguir as instruções encontradas no arquivo readme.txt na versão executável.
